# 🗂️ Django CRUD App with Authentication

## About

In this module, students build a full-stack web app using **Django**.
They will create and manage Authors and Books, connect data, and add user authentication.

This reflects real-world systems like internal dashboards in companies (e.g., managing employees and reports).
