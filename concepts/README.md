# CONCEPTS

**Learning objective:** Understand how a Django app works from database to UI.

---

### What is Django?

Django is a framework that helps you build web apps quickly.

It uses:

* **Models** → database
* **Views** → logic
* **Templates** → UI
* **URLs** → navigation

---

### Real-world example

Think of a company system:

* Authors → employees
* Books → reports/products

One employee can create many reports.

---

### Key idea: Flow of data

1. User visits URL
2. URL calls a view
3. View gets data from model
4. Template shows data

---

> 💡 Django handles many things for you (forms, login, security)
